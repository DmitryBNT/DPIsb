Задача: Определение производителя устройства

Решение: С помощью данных полученных из .pcap файла получить данные касательно mac-адреса устройства.
После получения данных о mac-адресе выполнение "сверки" кодировки производителя по списку достпуных кодировок производителей.
При совпадении определение производителя.

Задача: Выделение user-agent устройств

Решение: С помощью данных полученных из .pcap файла получить данные касательно user-agent устройств.
Выгрузка данных из протокола HTTP, и поиск требуемых параметров, если таковые там присутствуют. 
Вывод данных, отражающих название и версию приложения, операционную систему компьютера* и язык.
* Корректная обработка даннных вида: CPU OS 10_2 like Mac OS X - вероятностные данные Mac OS X.
