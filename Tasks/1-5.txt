Задача: Определение ОС узла сети и версии ОС (при возможности)

Решение: С помощью данных полученных из .pcap файла получить данные по ОС и вресии ОС. 
Выгрузка данных из протокола HTTP, и поиск требуемых параметров, если таковые там присутствуют. 
Корректная обработка даннных вида: CPU OS 10_2 like Mac OS X - вероятностные данные Mac OS X.
